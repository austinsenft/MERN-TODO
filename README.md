# MERN-TODO 
MERN Todo app with Login and Authentication. Vite, React-Query, React Router 6, MongoDB
Demo of functionality. Used Postman to verify updates and deletes

To run: 
`npm i`

Navigate to server
`node src/index.js`

Navigate to client 
`npm run dev`
